# TP1 - Simulation Interactive et Collecte de Données

## Binôme
- **Basma EL MGHARI**
- **Assmaa AZAROUAL**

## Objectif
Ce projet consiste à créer une scène 3D interactive sous Unity 6.4 permettant à un utilisateur de se déplacer et d'interagir avec des objets. Le système enregistre les actions de l'utilisateur (clics, trajectoires) dans un fichier CSV pour analyse, tout en intégrant un comportement IA simple (objet suiveur).


## Description
- Scène 3D interactive avec déplacements WASD + souris
- Interaction par clic sur objets (changement de couleur + logging)
- Collecte de données utilisateur (clics, temps, positions) export CSV
- Bonus IA : Objet suiveur intelligent


## Installation & Lancement

1. Ouvrir **Unity Hub**.
2. Cliquer sur **"Add"** → Sélectionner le dossier racine du projet (`TP1_Simulation/`).
3. Ouvrir la scène située dans `Assets/Scenes/SampleScene.unity`.
4. Appuyer sur le bouton **Play** (▶️) dans l'éditeur.

### Contrôles :
- **Déplacement** : Z, Q, S, D (ou W, A, S, D)
- **Caméra** : Souris
- **Interaction** : Clic Gauche sur les cubes/sphères
- **Sauvegarde CSV** : Touche `S`
